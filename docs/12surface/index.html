Fusion 360 N°4

Módulo de Superficies

Como 4to entregable en Fusion 360 se trabajó con el módulo de superficie, donde en vez de deformar un volúmen, se arman figuars a partir de planos, que proceduralmente pueden definir la forma futura.

Construcción

En primera instancia se traza una figura en un sketch, que será extruido para dejar una superficie. Luego se generaron dos superficies que definen los límites de altura de la figuara, al mismo tiempo que su curvatura es editable. El siguiente paso es aplicar un trim para eliminar los excedentes, para poder aplicar un stitch y unir las caras. Después se generó un sketch que corta la figura de manera horizontal, moviéndolo hacia adentro para poder generar hendiduras. Al cortar la figura, también se escaló y generó un loft entre los dos lados del corte, y luego se hizo un stitch para generar una sola figura. Finalmente se hizo un fillet en las esquinas y se añadió material al modelo.

Renderizado

Para la siguiente etapa se importa el modelo en keyshot, se añaden texturas, y opcionalmente se puede hacer un fillet en keyshot con las herramientas que aparecen abajo a la derecha en la imagen. Por último se renderiza la escena.
