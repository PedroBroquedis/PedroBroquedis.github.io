Spoofing

El primer trabajo consta de una situación teórica donde se proponen diversos efectos en el sector de Goya, Madrid, en el caso de que dejara de funcionar la localización satelital por el efecto del spoofing. Spoofing es cuando un sistema de localización es"engañado" al simular y alterar la posición mostrada por los dispositivos de localización. Para este caso particular, se estudió el sistema de localización Galileo y todos los programas que lo rodean. Éste, fue creado por la ESA con fines comerciales y se encargará de la localización con una mayor precisión a nivel mundial. Algunos de los sistemas explorados fueron: funcionamiento de un reloj atómico, otros sistemas de localización (sistema americano, ruso y chino), funcionamiento de los sistemas de georreferenciación, usos para la localización satelital, y casos de spoofing reconocidos (principalmente usados por Rusia para el encubrimiento y seguridad de Putin). Luego se decidió un sitio de alto tránsito en Madrid, ya que actúa como una de las sedes operadoras del sistema Galileo. Al decidir este sitio, se propuso la situación teórica donde se crean torres de spoofing, y como el sector de Goya se podría adaptar a estos nuevos sistemas, donde la localización no es la real, si no que una simulada(similar al efecto que entrega un VPN).

Se muestra el proyecto y un mapa de estudio completo

Para esto se crean sistemas como:

  1. Embajadas
  2. Sistemas de Entregas
  3. Cines que pasen por el bloqueo de región
  4. Mercados
  5. Comunidades Anti-Tecnología
  6. Granjas de Hackers
  7. Hoteles con horarios diferenciados
  8. Torres de Spoofing

Habitáculo

Como segundo proyecto, se presenta la creación de un habitáculo localizado dentro del edificio D de la Universidad Adolfo Ibañez. Para este proyecto se crearon una serie de placas de acrílico hexagonales, unidad por una cuerda de polipropileno. De esta manera se generó una superficie que quedó suspendida en el aire por medio de anclajes. Para el origen de la idea, las placas fueron basadas en la elaboración de las armaduras de los guerreros samurai, específicamente del “kozane” (pequeñas placas unidas por cuerdas de seda), para así escalarlo y modificar la forma de este mismo.
