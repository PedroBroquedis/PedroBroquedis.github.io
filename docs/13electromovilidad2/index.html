Electromovilidad

En esta unidad se estudiará la electromovilidad, es decir el transporte y movimiento dentro y fuera de las ciudades mediante vehículos eléctricos.

Sketches Iniciales

En la primera fase, se definió un problema, que en este caso es el no uso del espacio de suelo. Luego se generaron sketches para solucionar este desafío, y se generaron 3 categorías: Drones personales, Transporte Urbano y Logística.
